AJAX不能独立工作。它与其他技术结合使用，可以创建交互式网页。

JavaScript

  • 松散类型的脚本语言。
  • 在页面中发生事件时调用JavaScript函数。
  • 整个AJAX操作的胶水。

DOM

  • 用于访问和操作结构化文档的API。
  • 表示XML和HTML文档的结构。

CSS

  • 允许将演示样式与内容清晰分离，并可通过JavaScript以编程方式进行更改

XMLHttpRequest的

  • 与服务器执行异步交互的JavaScript对象。
